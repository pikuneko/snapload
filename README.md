# SnapLoad

SnapLoad は、YouTube ビデオとオーディオのダウンロードを支援するウェブアプリです。

## 更新履歴

### v1.4.0 - 2026.01.29
#### Security Hardening & Enhanced Protection 🔒
- セキュリティヘッダー強化：XSS、CSRF、クリックジャッキング対策を実装。
- レート制限機構の導入。ブルートフォース攻撃は15分で最大5回まで。
- ログ情報の機密化。User-Agent、ファイルパス、Cookie情報を自動マスク。
- 入力値バリデーション強化。YouTubeのURL検証、パスコード長チェック（最小4文字）。
- タスクID生成の安全性向上。UUID ベースに変更し、予測不可能に。
- セッションタイムアウトを1時間に短縮、SameSite=Strictで CSRF 攻撃を徹底防止。
- エラーメッセージの詳細情報を隠蔽し、情報開示を最小化。

### v1.3.0 - 2026.01.06
#### Usability Boost & History Management 🛠️
- SnapBird ハイスコア機能の実装。記録を保存できるようになりました。🏆
- 履歴の削除機能を追加。不要な履歴を整理できます。🗑️
- 履歴検索機能を履歴画面に追加。目的のファイルをすぐに見つけられます。🔍
- ダウンロード中の速度と残り時間の表示を改善（より正確な情報を提供）。⚡
- 全体的なマイクロアニメーションとUIの使い勝手をブラッシュアップ。

### v1.2.0 - 2026.01.06
#### UI/UX Major Overhaul ✨
- AOS (Animate On Scroll) を導入し、流れるようなアニメーションを実装。
- iTyped.js によるスタイリッシュなタイピング演出をヘッダーに追加。
- SweetAlert2 による法的同意（利用規約・Cookie）ダイアログを実装。
- モダルなガラスモーフィズムデザインの更なるブラッシュアップ。

### v1.1.5 - 2026.01.05
#### Functionality & Core Fixes 🛡️
- プレイリストダウンロードの完全サポート（ZIP圧縮機能付き）。
- ダウンロード履歴のファイル再利用ロジックを最適化。
- MP3/MP4の自動判別バッジの修正。
- サーバーログのリアルタイム表示機能の強化。

### v1.0.0 - 2026.01.01
#### Official Release 🚀
- SnapLoad プロジェクト始動。
- 高画質YouTubeビデオ・オーディオダウンロード。
- パスコード保護機能の実装。
- 履歴管理システム（JSONベース）の構築。
